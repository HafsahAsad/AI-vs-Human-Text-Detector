# AI vs Human Text Detector

This project is designed to detect whether a piece of text was written by an AI or a human using Machine Learning techniques.

## Features

- Uses natural language processing (NLP) to analyze text patterns
- Classifies text as AI-generated or human-written
- Built with Python and common ML libraries
